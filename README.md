## How to install

You will need:

- Unity3D (2017.1 or higher advised).
- A clone of this repository.

Steps to build:

- Download / clone this repository
- The Play.unity scene is the starting point (main menu)
- Ensure the platform is set to `PC, Mac and Linux Standalone`
- Click Build!
